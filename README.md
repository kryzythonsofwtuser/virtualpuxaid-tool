# VirtualPuxaid Tool 🚀

### 💻 Kali Linux
```bash
git clone https://github.com/kryzythonsofwtuser/virtualpuxaid-tool.git && cd virtualpuxaid-tool && pip install -r requirements.txt && python virtualpuxaid.py
```

### 📱 Termux (Android)
```bash
pkg update -y && pkg upgrade -y && pkg install -y python git termux-api && git clone https://github.com/kryzythonsofwtuser/virtualpuxaid-tool.git && cd virtualpuxaid-tool && pip install -r requirements.txt && python virtualpuxaid.py
```

> ⚠️ Instale o app **Termux:API** via F-Droid.
